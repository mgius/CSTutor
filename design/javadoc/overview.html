The CSTutor software system provides functionality to view lessons created by an instructor. The system is comprised of three application programs:

  • the Calendar Tool program, for regular users to manage and view their calendars
  • the Calendar Tool Administration program, for administrators to manage databases and control other functions of Calendar Tool central hosts
  • the Calendar Tool Server, which runs in the background on central hosts to control communication among users' local computers and the host
The requirements specification document has complete details of calendar tool functionality.
