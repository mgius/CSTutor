The CSTutor software system provides functionality to view, edit, and create courses, lessons, and quizzes for an instructor. In addition, students can take quizzes and view lessons that are created by an instructor. The system is comprised of two application programs:

  • the CSTutor website, for students can view lessons and take quizzes, and instructors can view, edit, and create courses, lessons, and quizzes. The website sends requests to the server for data.
  • the CSTutor Server, accepts requests from users and responds with HTML and generates pages for the website.
The requirements specification document has complete details of CSTutor functionality.
