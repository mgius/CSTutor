While Loops

For loops are a type of loop that have a single conditional statement. While the statement is true, the loop body will execute. After each iteration of the loop, the condition is checked again. A true condition runs the loop body again, and a false condition end the loops and the code after the loop body is executed.

Unlike a For Loop, the "count" expression is not an intrinsic part of the loop. You must structure the loop so that the conditional statement will eventually evaluate to false. 

int x = 0;
while (x < 10) {
   System.out.println(x);
   x++;
}

Note that without the "x++" statement within the loop body, this loop would never end.
