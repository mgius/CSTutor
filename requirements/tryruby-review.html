TryRuby

TryRuby is a web application that simulates a Ruby shell, with a nice builtin tutorial.  Originally written by why the lucky stiff, the project was taken over by Andrew McElroy and David Miani after why's disappearance.

Good Features

  • Web-based - no need for the user to install and configure Ruby
  • Chapters in the tutorial
  • Ability to move backwards in the tutorial
  • Can reset the session to give a fresh start
  • Acts just like a session in irb, the Ruby shell
  • Instructions have to be followed successfully to get to the next page
  • Instructions are simple and bite-sized

Bad Features

  • Only two chapters
  • Slower than using irb directly
  • No table of contents
  • No ability to skip to a certain part of a chapter
  • Only covers the basics of Ruby
