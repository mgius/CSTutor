1.5.4. OpenJS Tutorial

OpenJS aims to teach JavaScript in an interactive way. The website layout is similar to how our layout would be; the main page has a side bar on the right with top level categories such as variables, arrays, if conditions…etc. Each of the aforementioned hyperlinks directs you to a page with a lesson about that particular topic.

Each page begins with a description of the syntax and some introduction explaining how to use it. Below the introduction is a text box containing code that demonstrates a specific example of how the code can be used. This text box has a “Run” button below it that executes the code in the text box. The text can be modified by the user so that they may get a better understanding of how the code works by playing with it.

Good Features:

  • Layout of lessons is on the side of the main page
  • Each lesson is well explained and includes sample code
  • Example code can be run and edited

Bad Features:

  • Nothing struck me as particularly bad

Missing Features:

  • The layout of lessons doesn't allow subcategories of lessons, preventing a hierarchal design
  • Interactive code lacks error reporting
  • Lacks any form of user progress tracking
  • Lacks quizzes
