1.5. Related Systems

Widely used commercial systems that provide functionality similar to the CSTutor include

  • Vimtutor (no Site available. Plaintext file installed with vim)

  • Sun's Java Tutorial

  • Eloquent Javascript Tutorial

  • OpenJS

  • w3schools SQL Tutorial

  • Sophrinix Ruby Tutorial

All of these systems provide a tutorial of a select programming languages. The goal of the CSTutor is to provide basic functionality found in the tutorials and to expand upon them. The areas we plan on improving are:

  1. providing in depth lessons meant to supplement class lecture:

  2. provide quizzes after each section to let students know how they are progressing and let instructors know if students are having any troubles:

  3. the CSTutor will be able to be used online and offline.

The goals for the CSTutor are not met by any other alternative currently on the market for a teaching environment.
