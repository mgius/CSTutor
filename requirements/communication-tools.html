2.5. Communication Tools

CSTutor is primarily intended to be used as a solitary study aid. However, the following methods of communication are integrated into CSTutor.

2.5.1 Class Chat

During any lesson, select the "Chat" Button (see 2.1 UI Overview) to open up the integrated IRC interface.

Figure 2.5-1: Web Based Chat Room Interface - Before connecting

The interface is initially unconnected. To connect, the user should press the "connect" button in the middle of the interface.

Figure 2.5-2: After connecting

Once the interface has loaded and connected, the interface behaves as a standard IRC chat room. The user should change their displayed name from the randomly generated one to a name of their choosing using the /nick command.

Figure 2.5-3: Changing displayed name

Now that the user has changed their name, they should send a message. To send a message, a user will type their message into the text field at the bottom of the chat interface, then press Enter (or Return). As other users post messages to the chat room, those messages will appear on every users screen.

Figure 2.5-4: Typing a message

Each Class will have its own chat room.
