2.3.4.2 Deleting a User from the Class Roster

This scenario shows the instructor of a class how to remove users from the class roster. Removing users is important because it allows the instructor to properly manage the class roster. In this scenario, we assume that the instructor has logged in, selected the appropriate class, and is viewing the roster page. Details of logging in as an instructor can be found in Section 2.1

To remove a user from the class roster, the instructor selects Delete User on the roster management screen and then the diaglog box shown in Figure 2.3.4.2-1 is displayed by the system.



Figure 2.3.4.2-1: Remove User Dialog.


The user field is a one line string where the user name to be deleted is typed in. After the user name is entered click Delete to delete the user from the class roster.
