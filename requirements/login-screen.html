2.3 Login Screen

This will introduce all functionality of the Login Screen.

2.3.1 Logging In

This scenario shows how a user (student or instructor) logs in.

The login screen (see Figure 1 below) is the first screen that a user will see upon opening CSTutor.

Figure 1: Initial Login Screen

The user may type in their Username and Password then click the Login button in order to login.

Once logged in the user will be presented with either the instructor page FIXME or the student page FIXME depending on their account.

2.3.2 Anonymous Login

A user may choose to log in anonymously by selecting the Anonymous checkbox. When this box is checked no user tracking information is recorded. In this case the user does not need to type in their username or password. The user must simply check the box and click login in order to login.

2.3.3 Registering a New Account

If a user wishes to create a new account the user must click the Register New Account button. The program will respond by opening the New Account Registration window (see Figure 2 below).

Figure 2: New User Registration

 Once in the New Account Registration window, the user must enter their desired Username into the username field. Next the user must type a password twice in order to verify it was a correct typing. When the user is satisfied with the entered information, clicking Submit will request the username from the server (details of how this process works not decided yet) and the window will close.
