3.2.1. Security and Privacy

Classes with a security setting of 'Private' must only be viewable by users with the 'View' permission for the given class. All user account passwords must be kept secure from any CSTutor user. All user account profile information must be kept secure from anyone other than the account owner and any users with the 'instructor' permission. All class-wide statistical information must only be viewable by instructors and students with the 'stats' permission, while each individual user can access their own statistics. All class pages and their associated options and answer keys must be kept secure from anyone except the class instructor and any student with the 'edit' permission.

3.2.2. Simplicity versus Power

On a spectrum of simple-to-use versus powerful, CSTutor is split depending on the user type. Instructors have some more powerful tools and are assumed to be computer literate and be knowledgeable on the use of computer networks consisting of remote users, local users, and central data servers. Student users see a much more simple interface and are only expected to be knowledgeable of the fundamentals of using a computer and the internet.
