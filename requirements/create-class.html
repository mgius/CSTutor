2.3.1.1 Creating a new Class

The "Create Class" button is only visible to instructors.

Classes are how lessons are grouped together for an insructor to manage. A class is always the top folder in a directory, and anything within the class is a lesson or quiz.
For an instructor to add a new class, he or she clicks on the "Create Class" button located in the Navigation bar. CSTutor displays the screen in Figure 2.3.1.1-1 and once completed, a new class is created.

Figure 2.3.1.1-1
The instructor types in the name of the class in the "Class Name" field and checks the the "Private" checkbox to only allow certain student users to access the class lessons. Once the instructor has entered the class name, they can submit their class.

Figure 2.3.1.1-2

 Once submitted, CSTutor shows the instructor a page edit screen of the main page of the class(More information can be found on the Editing Lessons Page). CSTutor also adds the class to their navigation bar and the instructor is then able to create lessons for that class.
