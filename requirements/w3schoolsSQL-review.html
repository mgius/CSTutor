1.5.5. w3schools SQL Tutorial

w3schools.com is a website that provides online tutorials for many programming languages. The SQL tutorial contains both basic and advanced commands. In addition, it shows the results of what a SQL command will do. There is also a web page that lets a user practice SQL commands by entering them into a text box. However, this feature could have been on every tutorial web page so a users can practice as they go through the tutorial.

The web page is a solid reference for SQL as well as a decent tutorial. The sections for the SQL commands are fairly detailed and give a good explanation of what the command will do. Furthermore, the tutorial will show you how to do the commands in different versions of SQL (MySQL, Oracle, MS Access, or SQL Sever). Unfortunately, the website is not split up into lessons.



Good Features:

  • Covers basic and advanced SQL commands

  • Allows the user to practice SQL commands

  • Can pass a certification test

  • Error checking of SQL commands


Bad Features:

  • Not organized into lessons

  • Can't practice SQL commands as you go

  • Missing Features

  • No quizzes or tests for specific section

  • No user history
