2. Functional Requirements

The primary functionality of the CS Tutor is to allow students to view lessons for a class that an instructor creates. Instructors create lessons through the lesson management screen. In addition, instructors can create quizzes that can track the students' progress.

Following an overview of the CS Tutor user interface, details of system use are presented in the following scenarios:

  • an individual user logging into the CS Tutor
  • an individual user browsing calendars in a variety of ways
  • an individual user scheduling other types of items
  • a group leader scheduling a meeting with several attendees
  • a system administrator updating the calendar databases
  • an individual user setting system options
  • details of file and edit commands
These scenarios cover all functional requirements for the system.
