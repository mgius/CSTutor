1. Introduction

This document defines requirements for an interactive electronic tutoring program called CSTutor. CSTutor allows users to learn programming lessons in an interactive manner. Instructors use tools within the program to create a hierarchical lesson design. Instructors may place runnable program code into the lessons. These sections of code allow users to edit and run the code interactively in order to better understand the language. CSTutor also provides functionality to create quizzes and to track user historical progress through quizzes and lessons. Students may check their own progress and Instructors may access all Students' progress and statistics. The program initially supports Java lessons but has functionality to import other languages.
