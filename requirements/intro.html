1. Introduction

This document defines requirements for an interactive electronic tutoring program called CSTutor. The program allows users to learn programming lessons in an interactive manner. Lesson creators use tools within the program to create a hierarchical lesson design. Individual lessons may have text boxes with example code that users may modify and run in order to better understand the language. The program also provides quizzes and tracks user's historical progress. Administrators or lesson planners have the ability to track student's progress and create more content. Lesson creators may also mandate that certain prerequisites are met in order to progress through the lesson.
