2.3.1.3 Editing a Class/Lesson

An instructor is taken to an Edit Lesson screen in two ways

1. The instructor has just created a new class or lesson and was taken this screen to fill in the main page. The initial screen in this case would be blank(For more information, see "Creating new Classes/Lessons").

2. The instructor was browsing his or her lessons and classes and clicked on the "Edit" button. The screen will be populated with the data for that lesson and will be editable by the instructor.

Figure 2.3.1.3-1
To edit an existing lesson, the instructor browses lessons in the navigation bar. Notice the selected lesson is highlighted. The data will show up as though the instructor was following the lessons. Once he or she finds the lesson, they first click on the topic to view it, then click on Edit to begin editing.


Preview/Save button: Clicking the Preview/Save button saves the current edits. The user is then taken to a screen that shows the lesson a student would view it. This does NOT make the changes visible to the students in the class.

Save button: Clicking the Save button save the current edits only. A save conformation will appear if save is completed.

Figure 2.3.1.3-2



Publish button: Clicking the Publish button save the current edits and publishes them to the class. Students will be able to view the changes after the lesson is published.

Figure 2.3.1.3-3



Import button: Clicking the Import button will import data into the lesson. Files that can be imported are: jpeg, gif, png, html, txt, java, c, cpp
Once the button is pressed, a file browser opens to select a file.

Figure 2.3.1.3-4



Options button: Clicking the Options button pushes a dropdown list of options onto the screen.

Figure 2.3.1.3-2


Modify Prerequisites: Clicking this allows the user to pick from a list of previous lessons or quizzes. Selecting an item means that the Student must have completed the lesson or quiz prior to viewing this page. When selecting a quiz the Instructor may select a minimum score required. See Figure 3 below.

Figure 2.3.1.3-3



Delete Lessons: Deletes the current lesson and returns the Instructor to a nearby lesson or home page if there are none.

Undo All Changes: This removes all changes from the working copy. The new working copy will be an exact copy of the currently published lesson.

Toolbox button: Clicking the Toolbox button places a toolbox onto the screen. It can be removed by clicking Toolbox button again.

Figure 2.3.1.3-4


Description of tools:
The box button allows the user to draw boxes.
The circle button allows the user to draw circles.
The line button allows the user to draw lines.
The IMG button allows the user to insert images.
The Code button allows the user to insert code boxes.
The Font button allows the user to change the font.
The Size button allows the user to change the text size.
The Align button allows the user to select left, center, or right text alignment.
