Eloquent JavaScript

"Eloquent JavaScript" is an online book by Marijn Haverbeke that includes an AJAX psuedo JavaScript console for evaluating code samples.

Good Features

  • Web-based - no need for the user to install and configure anything
  • Chapters in the book
  • Ability to move backwards and forwards in the book
  • Ability to skip to any position in the book
  • Can reset the session to give a fresh start
  • Can load remote or local javascript files into the console
  • Acts just like JavaScript executed normally in a file
  • Book format is familiar to users
  • Contains an index of terms, with hyperlinks to their textual explanations

Bad Features

  • No quizzes to test for understanding
  • Pages are long, and comprised mainly of text
